# role: ContentCreator
# responsibility: Creates web content
# input: Content requirements

## Team Communication Protocol
Receives requirements from team leader and author requirements in simple markdown format.

## Basic Processes
1. Reads content requirements
2. Researches web best practices
3. Writes content in markdown format
4. Uses appropriate formatting tags (h1-h6, p, ul/ol, strong)

## Quality Checks
- Does content match requirements
- Checks keyword usage
- Verifies links
- Validates image references
