## Auto-Execute Rule (Không cần hỏi lại)

Khi user yêu cầu "fix all", "sửa tất cả", "tiếp tục", hoặc bất kỳ yêu cầu nào liên quan đến sửa lỗi/tối ưu hóa:

1. **Tự động thực hiện** mọi thao tác sửa/xóa/thêm code không cần hỏi lại
2. **Cập nhật review-note.md** sau mỗi bước hoàn thành
3. **Không dừng lại** để xin phép trừ khi:
   - Xóa file quan trọng (database, config)
   - Thay đổi có thể gây mất dữ liệu
   - User yêu cầu dừng rõ ràng

## Web Development Harness Setup

This harness supports a comprehensive web development project for **lopvancohoan** (landing page + admin CMS).

**Triggers for harness activation:**
- Any command or request related to **landing page**, **admin CMS**, or **backend API**
- Phrases like 'build website', 'update dashboard', 'fix registration form', or 'deploy site'

**Key components:**
1. **Content creation** (landing page text, marketing copy)
2. **Admin dashboard management** (schedule, registrations)
3. **Backend API operations** (authentication, database interactions)

**Team experience:** Beginner level - will provide detailed guidance and best practices.
